# linux搜索文件

## find
find 目录 条件 动作
'''Shell
find / -name 'interfaces'
'''
在根目录下查找名字为interfaces所在位置
## locate
locate 比find要快一些，因为它查找的/var/lib/locatedb
'''Shell
locate interfaces
'''
## whereis
whereis查找可执行文件 也就是二进制文件
'''Shell
whereis grep
'''
## which
which查看系统命是否存在，并返回命令所在位置
'''Shell
which grep
'''
输出
'''Shell
/bin/grep
'''
## type
 type查看是否是系统自带指令
'''Shell
type cd
'''
